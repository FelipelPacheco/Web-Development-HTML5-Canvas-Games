# Projeto Breakout

Este é um jogo de estilo Breakout. O projeto foi construído utilizando a biblioteca **p5.js** e a linguagem **JavaScript**.

## 👥 Integrantes
Felipe Luiz Pacheco

## 🎮 Como Jogar
Para testar o jogo, rodar o código no (https://p5js.org).

### Controles:
* **ENTER**: Inicia o jogo a partir do menu principal.
* **Setas (Esquerda/Direita)**: Movimentam a raquete.
* **Tecla R**: Reinicia o jogo após o Game Over ou Vitória.

## 🚀 Funcionalidades
* **Menu de Início**: Com título e instruções.
* **Sistema de Pontuação**: Cada bloco destruído soma 10 pontos.
* **Condição de Vitória**: Quebrar todos os blocos existentes
* **Game Over**: O jogo finaliza caso a bolinha ultrapasse a linha da raquete.
